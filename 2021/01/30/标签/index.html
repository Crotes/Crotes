标题标签

基本内容

网页标题分为6个等级，< h1 > - < h6 >

h为head的缩写，意思为头部，标题

语法

标签语义

作为标题使用，依据重要性递减

特点

1.加了标题的文字会加粗，字号变大

2.一个标题独占一行，后面的文字会在下一行显示

段落标签

基本内容

< p >标签用于定义段落，它可以将整个网页分为若干段落

p为paragraph的缩写，意为段落

语法

标签语义

把HTML文档分割为若干段落

特点

1.文本在一个段落中会根据浏览器窗口的大小自动换行

2.段落与段落之间含有空隙

换行标签

基本内容

在HTML中，一个段落中的文字会一直到浏览器右边才会换行，如果要强制换行，需要**换行标签< br / >**，这是一个单标签

b为break的缩写，意为打断

语法

标签语义

强制换行

特点

1.< br / >是一个单标签

2.< br / >标签只是简单地开始新的一行，跟段落不一样，段落之间会插入一些垂直的间距

文本格式化标签

基本内容

在网页中有时需要为文字设置粗体，斜体，下划线等效果，这时需要用到HTML中的文本格式化标签，使文字以特殊方式显示

标签语义

突出重要性，比普通文字更重要

语法

语义 标签 说明
加粗 < strong>< /strong>或< b>< /b> 更推荐使用< strong>标签加粗，语义更强烈
倾斜 < em>< /em>或< i>< /i> 更推荐< em>标签
删除线 < del>< /del>或< s>< /s> 更推荐< del>标签
下划线 < ins>< /ins>或< u>< /u> 更推荐< ins>标签

< div >和< span >标签

基本内容

< div >和< span >没用语义，他们就是一个盒子，用来装内容

div是division的缩写，表示分割，分区

span意为跨度，跨距

语法

特点

1.< div>标签用来布局，但是现在一行只能放一个< div>。

2.< span> 标签用来布局，一行上可以放多个< span>。

图像标签与路径

图像标签

基本内容

< img>标签用于定义HTML页面中的图像

src是< img>标签的必须属性，它用于指定图像文件的路径和文件名

语法

图片属性

属性 属性值 说明
src 图片路径 必须属性
alt 文本 替换文本，图像不能显示的文字
title 文本 提示文本，鼠标放在图像上的文字
width 像素 宽度
height 像素 高度
border 像素 边框粗细

注意点

1.图像标签可以有多个属性，必须写在标签名后面

2.属性之间不分前后，但必须隔开

3.属性采取键值对的格式，key = “value”

路径

路径可以分为：

相对路径：以引用文件所在位置为参考基础，建立出的目录路径

相对路径分类 符号 说明
同一级 图像文件位于html文件的同一级
下一级 / 图像文件位于html文件的下一级
上一级 ../ 图像文件位于html文件的上一级

绝对路径：目录下的绝对位置，直接到达目标，通常从盘开始，注意符号为\

超链接标签

基本内容

< a>标签用于定义超链接，作用是从一个页面链接到另一个页面

分类

1.外部链接 例：< a href= “https://www.baidu.com">百度< /a>

2.内部链接：网站内部页面之间的相互链接，直接链接内部页面名称即可

3.空链接：如果当时并未确定链接目标时，< a href = “#”>首页< /a>

4.下载链接：如果href里面地址是一个文件或压缩包，会下载这个文件

5.网页元素链接：在网页中的各种网页元素，如文本，图像，表格等

6.锚点链接：当我们点击链接，可以快速定位到页面中的某个位置

​ · 在链接文本的href属性中，设置属性值为#名字 的形式，如< a href = “#two”> 第二季 < /a>

​ · 找到目标位置标签，里面添加一个id属性 = 刚才的名字， 如

第二季介绍< /h3>

语法

属性

属性 作用
href 用于指定链接目标，当标签有了href时，就具有了超链接功能
target 用于指定链接页面的打开方式，_self为默认值， _ blank 为在新窗口中打开

注释标签

基本内容

添加便于阅读和理解但不需要再页面中的注释文字，就需要使用注释标签

HTML中的注释以“< !–” 开头，以” –>“结束

快捷键 ：ctrl + /

添加注释是为了更好地解释代码的功能，便于相关人员理解和阅读代码
